1. Go to https://replit.com/
2. Create a new Node.js repl.
3. In the `package.json` file, add the following dependencies:
   "express": "^4.19.2",
   "puppeteer": "^22.12.1",
   "puppeteer-stream": "^3.0.0",
   "ws": "^8.18.0",
   "node-fetch": "^2.0.0"
4. In the `server.js` file, copy the contents of the `server.js` file from this project.
5. In the shell, run `npm install`.
6. Run the server.
7. Copy the public URL of the server.
8. In the `index.html` file, update the WebSocket URL to point to the Replit server. For example: `const ws = new WebSocket('wss://<your-replit-url>');`
9. Also update the video stream url: `stream.src = 'https://<your-replit-url>/stream';`
